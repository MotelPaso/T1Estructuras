# T1Estructuras

### Instrucciones para compilar y correr el codigo

El codigo fue trabajado utilizando Arch y Ubuntu Linux, por lo tanto, las instrucciones serán para el compilador **gcc**.

Si se desea revisar utilizando otro sistema operativo, se puede utilizar el archivo `main` del repositorio con los siguientes comandos:
```
git clone https://github.com/MotelPaso/T1Estructuras
cd T1Estructuras
./main
```
